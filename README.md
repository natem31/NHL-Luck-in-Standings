# NHL-Luck-in-Standings
This project was a fun analytics project I created to observe how much a team deserves their place within the NHL Standings for the 2025-26 season. By using analytical tools such as linear regression, and advanced statistics such as xGdiff and PDO, I created an aggregate formula to observe how much teams are over or underperforming their expected outcomes.
My central question came from observing the difference in playoff race outlooks from the Eastern and Western conferences. This led me to see if i could in some way quantify the luck associated with each teams outcomes.
